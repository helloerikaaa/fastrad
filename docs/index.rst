Welcome to fastrad's documentation!
===================================

**FastRadiomics (fastrad)** is a GPU-accelerated Python library for calculating radiomics features from medical images.

It aims to be a high-performance, strictly-compliant alternative to PyRadiomics, built from the ground up to achieve identical feature extraction parity with significantly faster execution times by utilizing PyTorch for large-scale tensor operations.
